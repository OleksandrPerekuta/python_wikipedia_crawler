# Projekt zaliczeniowy z przedmiotu PPY (semestr: 4, rok: 2024).
## Wikipedia Crawler

This is a simple web crawler that searches for a path between two wikipedia pages.

Ui is implemented using Flet.

To run the app:

```
flet run 
```
